# 🦐 小蝦技能儀表板

OpenClaw 小蝦的 57 個技能學習進度儀表板。

## 檔案結構

```
skills-dashboard/
├── index.html         ← 儀表板主頁（含資料）
├── canvas-test.html   ← Canvas 基本測試頁
└── README.md
```

## 本機預覽

直接在瀏覽器開啟 `index.html` 即可。

## 資料更新

技能資料在 `index.html` 的 `<script>` 中的 `skills` 陣列，直接修改後重新整理即可。
